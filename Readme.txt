PHP/Laravel project.

Be sure to have all the requirements installed (check requirements.txt).

To launch the project :

1- Open a command prompt
2- Go on the project folder
3- Type composer install
4- Open another command prompt and go to the project folder
5- Type in one command prompt the following command : npm run dev
6- Type in the other command prompt the following command : php artisan serve
7- Click on the http://127.0.0.1:8000 link that will be displayed after the php artisan serve command

Enjoy !
